# Getting Started

## What is Alpi Maps?

Alpi Maps is an open-source outdoor map application designed for hikers, cyclists and adventurers. It combines powerful online tile streaming with full offline capability, so the same rich feature set is available whether you have a mobile signal or not.

Key capabilities at a glance:

- 50+ map styles streamed online or from local data packs
- Full offline vector maps, search, routing and elevation
- Turn-by-turn navigation for walking, cycling and driving
- Elevation profiles with ascent/descent statistics
- Astronomy tools, peak finder, GPS satellite view
- 100% privacy-first — no account, no telemetry, no ads

## First Launch

When you first open Alpi Maps you will see the map screen. By default it connects to online map providers so an internet connection is helpful for the initial view, but the app can run entirely offline once you add local data packs.

### Online mode

In online mode the app streams map tiles from the configured providers. You can:

1. Pan and zoom the map freely.
2. Tap the **search bar** to search for any place worldwide.
3. Tap any location on the map to get its address and elevation.
4. Tap the **GPS button** (bottom-right) to centre the map on your location.

### Setting up offline mode

For full offline operation, you need to add MBTiles data packs to your device. See the [Offline Maps guide](/guide/offline-maps) for detailed instructions.

## Basic Navigation

| Action | Result |
|---|---|
| Tap map | Select location / show info panel |
| Double-tap | Zoom in |
| Pinch | Zoom in/out |
| Two-finger drag | Tilt map (if pitch is enabled) |
| Rotate with two fingers | Rotate map (if rotation is enabled) |
| Tap GPS button | Centre on your location |
| **Long-press GPS button** | Start continuous location tracking |
| Tap Layers button | Open layers / style menu |
| **Long-press Layers button** | Quickly switch map style |
| Tap ⋮ menu button | Open map options |
| **Long-press ⋮ menu button** | Open map options panel |

## The Info Panel

Tapping any point on the map opens the bottom info panel. From there you can:

- See the address, elevation and coordinates
- Save the location as a marker
- Get routing directions to/from the point
- Open an elevation profile (for routes)
- Check weather at that location (requires OSS Weather app)
- View astronomical data (sunrise, sunset, moon)
- Search the web for the location name
- Open Wikipedia for the selected place
- Share the location
